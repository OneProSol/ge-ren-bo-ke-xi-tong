### 一、相关技术
- 前端：`thymeleaf`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`
- 适配：手机端和pc端网页

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 首页、2. 博客详情、3. 评论

管理员功能：1. 登录、2. 首页、3. 发布博客、4. 博客管理、5. 博客修改、6. 评论管理、7. 分类管理、8. 标签管理、9. 系统配置、10. 修改密码
